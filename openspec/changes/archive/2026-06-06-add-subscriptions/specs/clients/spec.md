## ADDED Requirements

### Requirement: Кнопка абонемента на карточке клиента

Бот **SHALL** на карточке активного клиента показывать кнопку абонемента: если у клиента нет
активного абонемента — кнопку создания абонемента; если активный абонемент есть — кнопку
перехода на его карточку, текст которой включает остаток встреч `remaining`. На карточке
архивного клиента кнопка абонемента **SHALL NOT** показываться.

#### Scenario: У активного клиента нет абонемента

- **WHEN** специалист открывает карточку активного клиента без активного абонемента
- **THEN** на карточке показана кнопка создания абонемента

#### Scenario: У активного клиента есть абонемент

- **WHEN** специалист открывает карточку активного клиента с активным абонементом, у которого `remaining = 5`
- **THEN** вместо кнопки создания показана кнопка перехода на карточку абонемента с остатком `5` в тексте

#### Scenario: Карточка архивного клиента

- **WHEN** специалист открывает карточку архивного клиента
- **THEN** кнопка абонемента на карточке не показывается
