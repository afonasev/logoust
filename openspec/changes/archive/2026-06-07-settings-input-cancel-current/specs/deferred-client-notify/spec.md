## ADDED Requirements

### Requirement: Отмена на шаге ввода своего времени

На шаге ввода «Своё время» отложенного уведомления бот **SHALL** предлагать
inline-кнопку «Отмена». Нажатие «Отмена» **SHALL** возвращать к экрану выбора момента
отправки (Сейчас / в пресет-время / Своё время), сохраняя контекст уведомления, и
**MUST NOT** ставить уведомление в очередь.

#### Scenario: Отмена возвращает к выбору момента

- **WHEN** специалист на шаге ввода своего времени нажимает «Отмена»
- **THEN** бот показывает экран выбора момента отправки уведомления
- **AND** ни одно уведомление не ставится в очередь
