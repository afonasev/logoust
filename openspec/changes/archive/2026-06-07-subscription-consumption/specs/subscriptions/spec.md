## MODIFIED Requirements

### Requirement: Карточка абонемента

Бот **SHALL** показывать специалисту карточку абонемента с датой создания, числом купленных
встреч `purchased`, остатком `remaining`, действиями (вычесть встречу, продлить, закрыть) и
**журналом списаний** — списком неотменённых строк списания отдельными кнопками (дата и, для
авто-списания, дата/время встречи), тап по которым открывает экран списания. На карточке
закрытого абонемента журнал показывается только для чтения.

#### Scenario: Просмотр карточки активного абонемента

- **WHEN** специалист открывает карточку активного абонемента
- **THEN** бот показывает дату создания, `purchased`, `remaining`, кнопки «вычесть встречу», «продлить», «закрыть», возврат на карточку клиента и журнал списаний кнопками

#### Scenario: Журнал списаний на карточке

- **WHEN** у абонемента есть неотменённые списания
- **THEN** бот показывает их отдельными кнопками; тап по строке открывает экран этого списания

#### Scenario: Отменённые списания скрыты

- **WHEN** у абонемента есть отменённое списание
- **THEN** оно не показывается в журнале на карточке

### Requirement: Ручное списание встречи

Бот **SHALL** по действию «вычесть встречу» уменьшать `remaining` активного абонемента на 1 и
оставлять строку в журнале списаний без привязки к встрече (`appointment_id IS NULL`).
Уменьшение остатка и создание строки журнала **MUST** быть атомарны. Если `remaining` уже равен
0, списание **MUST NOT** выполняться (в минус остаток не уходит) и строка журнала **MUST NOT**
создаваться, абонемент остаётся `active`. Комментарий после встречи к ручному списанию
добавляется позже на экране списания.

#### Scenario: Успешное списание

- **WHEN** специалист на карточке абонемента с `remaining = 5` нажимает «вычесть встречу»
- **THEN** `remaining` становится 4, статус остаётся `active`, в журнале появляется строка без привязки к встрече
- **AND** карточка обновляется с новым остатком

#### Scenario: Списание при нулевом остатке

- **WHEN** специалист нажимает «вычесть встречу» на абонементе с `remaining = 0`
- **THEN** `remaining` не меняется и остаётся 0, строка журнала не создаётся
- **AND** бот сообщает, что списывать нечего
