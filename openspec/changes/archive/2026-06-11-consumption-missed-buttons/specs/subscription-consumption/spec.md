## MODIFIED Requirements

### Requirement: Отчёт специалисту по итогам прохода

Бот после авто-прохода **SHALL** отправлять специалисту отчёт, если есть что показать: список
затронутых абонементов кнопками (имя ребёнка + остаток, переход на карточку абонемента) и
список прошедших встреч без списания — тоже **кнопками** с пометкой ❗ (имя ребёнка, время и
причина в подписи). Кнопка ❗-встречи **SHALL** вести: при причине «активного абонемента нет» —
на карточку клиента; при причине «абонемент закончился» — на карточку этого абонемента. ❗-встречи
**MUST NOT** дублироваться текстовыми строками в теле сообщения (заголовки блоков остаются
текстом). Если за проход не было ни списаний, ни ❗-встреч, отчёт **MUST NOT** отправляться (день
при этом всё равно помечается обработанным). Сбой доставки отчёта **MUST NOT** откатывать списания.

#### Scenario: Есть списания

- **WHEN** проход списал встречи с одного или нескольких абонементов
- **THEN** специалист получает отчёт с кнопками на затронутые абонементы; тап открывает карточку абонемента

#### Scenario: ❗-встреча без активного абонемента — кнопка на клиента

- **WHEN** в отчёте есть прошедшая встреча клиента без активного абонемента (причина «активного абонемента нет»)
- **THEN** она показана кнопкой с пометкой ❗, именем ребёнка, временем и причиной в подписи
- **AND** тап по кнопке открывает карточку этого клиента

#### Scenario: ❗-встреча с законченным абонементом — кнопка на абонемент

- **WHEN** в отчёте есть прошедшая встреча клиента с активным абонементом `remaining = 0` (причина «абонемент закончился»)
- **THEN** она показана кнопкой с пометкой ❗, именем ребёнка, временем и причиной в подписи
- **AND** тап по кнопке открывает карточку этого абонемента

#### Scenario: Пустой вечер

- **WHEN** за проход не было ни списаний, ни ❗-встреч
- **THEN** отчёт не отправляется, но день помечается обработанным

#### Scenario: Сбой доставки отчёта

- **WHEN** доставка отчёта специалисту не удалась (он заблокировал бота)
- **THEN** выполненные списания не откатываются, день остаётся помеченным обработанным
